# Supported Control Codes

## CSI Control Codes

| Control Code | Name | Description                                            | Implemented |
| ------------ | ---- | ------------------------------------------------------ | ----------- |
| CSI Ps D     | CUB  | Cursor Backward [Ps] (default = 1)                     | ✅          |
| CSI Ps G     | CHA  | Cursor Character Absolute [column] (default = [row,1]) | ✅          |
